# Healthcare Analytics Dashboard (Tableau + BigQuery)

## 📊 Project Overview
This project analyzes healthcare data to identify trends in:
- Patient volume
- Cost analysis
- Hospital performance
- Doctor workload
- Insurance distribution

## 🛠 Tools Used
- Google BigQuery (Data Warehouse)
- Tableau (Visualization)
- SQL

## 📁 Data Source
Healthcare dataset with 50,000 records including:
- Admission & Discharge Dates
- Medical Conditions
- Medication
- Hospital, Doctor, Insurance Provider
- Billing Amount

## 📈 Key Features
- Year-based filtering
- medical condition filtering
- Cost trend analysis
- Top hospitals & doctors
- Medication vs Test Result heatmap
- Hospital vs Insurance analysis

## 🧠 Data Model
- Fact Table: health_data
- Views:
  - Yearly Cost Summary
  - Hospital vs Insurance
  - Doctor Performance
